# Syntax

| Token | Use |
| --- | --- |
| #! | Shebang (`#! [wallet address]`)
| # | Start of line buffer |
| ~ | Start of comment |
| + | Add |
| - | Subtract |
| * | Multiply |
| / | Divide |
| % | Modulo |
| ^ | Power |
| and, && | Boolean AND |
| or, \|\| | Boolean OR |
| xor, \|\|> | Boolean XOR |
| not, ! | Boolean NOT |
| [ ] | List |
| { } | Object |
